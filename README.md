# Bulb On/Off Switch

The Bulb On/Off Switch is a simple beginner-friendly JavaScript project that demonstrates DOM manipulation by switching a bulb image ON and OFF using buttons.

# Features

1. Turn bulb ON and OFF with a button click

2. Demonstrates JavaScript functions and DOM access

3. Clean and minimal UI

4. Beginner-friendly project

# Technologies Used

1. HTML5 – Page structure

2. CSS3 – Basic styling

3. JavaScript – Bulb toggle functionality

# How It Works

1. The Turn On button changes the bulb image to ON

2. The Turn Off button changes the bulb image to OFF

3. JavaScript updates the src attribute of the image

# Learning Outcome

1. Understanding JavaScript functions

2. Using getElementById()

3. Handling onclick events

4. Manipulating HTML attributes dynamically
